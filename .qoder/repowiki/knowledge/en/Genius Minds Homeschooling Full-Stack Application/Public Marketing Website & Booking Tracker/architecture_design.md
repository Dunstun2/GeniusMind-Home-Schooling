Flat collection of standalone HTML documents served directly by the web server — no build step, no framework. Each page (index.html, about.html, courses.html, blog.html, blog-post.html, contact.html, faq.html) is self-contained: it links the shared stylesheet `styles.css?v=1.4`, Google Fonts (Outfit + Inter), and three client scripts (`analytics.js`, `banner-cta-handler.js`, `script.js`). The navbar and footer are injected at runtime via `<div id="navbar-placeholder">` / `<div id="footer-placeholder">` placeholders, implying a separate server-side include or JS-driven partial loader lives outside this module. SEO is baked in through per-page meta tags, Open Graph/Twitter cards, canonical URLs, and inline JSON-LD schema (e.g. `EducationalOrganization`, `FAQPage`). The only interactive sub-app is `public/track.html`, a dark-themed single-page tracker that reads a `?token=` query param, calls `/api/booking/track/{token}` to render booking details and a threaded chat, supports file attachments via FormData, and auto-refreshes every 15 seconds. All assets (images, fonts) are referenced under absolute paths rooted at `https://geniusminds.website/`.
