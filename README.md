# Genius Minds Homeschooling

A modern, high-performance, and responsive landing page for **Genius Minds Homeschooling**, an online and in-person tutoring service based in Nairobi, Kenya.

## 🚀 Features

- **Premium UI/UX Design:** Built with a modern aesthetic, utilizing glassmorphism, smooth animations, and a polished layout.
- **Dark/Light Mode:** Integrated theme toggle for a tailored user experience.
- **Responsive Layout:** fully optimized for Desktop, Tablet, and Mobile devices.
- **Dynamic Services:** Clear pathways for Online Learning, Home-Based Tutoring, and Physical Location options.
- **Lead Generation:** Includes a functional, aesthetically pleasing booking/contact form ready to be hooked up to a backend.
- **SEO Ready:** Proper semantic HTML and metadata for improved search engine visibility.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure and accessible elements.
- **CSS3:** Custom properties (variables), Flexbox, CSS Grid, and media queries (No external UI frameworks).
- **JavaScript (Vanilla):** Theme toggling, smooth scrolling, and mobile navigation interactions.

## 💻 Getting Started

To view or modify this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/GeniusMind-Home-Schooling.git
   ```
2. Navigate into the directory:
   ```bash
   cd GeniusMind-Home-Schooling
   ```
3. Open `index.html` in your favorite web browser or use a tool like VS Code Live Server to view the site.
